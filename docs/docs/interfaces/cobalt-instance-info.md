# `CobaltInstanceInfo`

Information about the Cobalt instance.

## `version`

Current version.

**Type:** `string`

## `url`

Server URL.

**Type:** `string`

## `startTime`

Server start time in Unix milliseconds.

**Type:** `string`

## `durationLimit`

Maximum downloadable video length in seconds.

**Type:** `number`

## `services`

Array of services which this instance supports.

**Type:** `string[]`
