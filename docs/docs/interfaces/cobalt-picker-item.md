# `CobaltPickerItem`

Item from a [`CobaltPicker`](cobalt-picker) response.

## `type`

Specifies what kind of item it is.

**Type:** `'photo'`, `'video'` or `'gif'`

## `url`

URL of the item.

**Type:** `string`

## `thumb` <Badge type="info" text="optional" />

Thumbnail URL of the item.

**Type:** `string`
